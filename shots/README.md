# Shots

`Shots` is the backup subsystem project for `shost`.

Name:
- `shost` backups are called `shots`
- individual backup jobs are `shots`
- the backup UI/module is `Shots`

This directory is intended to be a focused Claude Code work area for building the first version of backup management inside the existing `shost` app.
